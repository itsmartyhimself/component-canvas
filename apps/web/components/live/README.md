# live

App components. Wrap imports from `components/imports/*` and apply visual styling here.

Rules:
- Tailwind for layout and behavior only (flex, grid, gap, positioning, display).
- All visual design (color, radius, spacing, typography) via CSS variables from `app/globals.css`.
- Typography via `.type-1`..`.type-13` classes. Never `text-sm` / `text-lg`.
- Zero stock Tailwind or shadcn visual fingerprints.
